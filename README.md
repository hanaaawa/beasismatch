# BeasisMatch 🎓

BeasisMatch adalah platform web berbasis Laravel untuk membantu mahasiswa menemukan dan mencocokkan beasiswa sesuai profil akademik secara real-time.

---

## 🚀 Fitur Utama

- 🔐 Login & Register (Email + Google OAuth)
- 🎯 Matching beasiswa berdasarkan profil user
- 📚 Manajemen data beasiswa
- 💬 Forum diskusi
- 👤 Profil pengguna
- 🧠 Sistem rekomendasi otomatis
- 🛠 Admin panel (Filament)

---

## 🧱 Tech Stack

- Laravel 11+
- PHP 8.3
- MySQL / MariaDB
- Docker (Nginx + PHP + DB)
- Filament Admin Panel
- Google OAuth (Socialite)
- Tailwind CSS
